The tables contain data according to version 2.0 of the GDP standard and can be linked using the applicationID column. Data on upcoming calls and ongoing calls that do not yet have any applications linked to them are not included as the presentation is based on applications. Other data content according to version 2.0 is not included:

• English titles of calls and applications
• English descriptions of calls and applications
• English names of research subjects and sustainability goals
• English names of organisations
• Corporate identity number for funding bodies
• Currency codes for amounts

The presentation in the tables is one row per application in "Application" and "Application classification" and one row per person in "Person". In "Decided funding", the presentation is one row per funded organization and calendar year for planned payments. When multiple values occur in one and the same cell, the values are separated by semicolons.

In the table “Application”, organisations with the role “Applicant Coordinator” or “Coordinator” are presented as coordinators. Organisations with the role “Co-applicant organisation” or “Funded organisation” are presented as co-applicant organisations if they do not have a coordinating role.

The sorting of the table is by application registration number (descending). The sorting of “Decided funding” is by application registration number (descending), calendar year of planned payment (ascending) and name of funded organisation (alphabetically). The sorting of “Person” is by application registration number (descending) and role for person (alphabetically). The sorting of “Application classification” is by application registration number (descending).
